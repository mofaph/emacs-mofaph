
Emacs 中一些有用的功能：

C-h k : 查找按键绑定的函数
C-h w : 查看函数是否已经绑定到按键
C-h f : 描述函数
C-h m : 描述当前的所用到的模式
C-h v : 描述变量
C-h p : 根据主题来查找扩展包
C-h P : 根据给出的包名显示扩展包的说明信息

C-h d : 查找符合给出的正则的命令和变量的文档
C-h F : 查找给出的命令的在线文档
C-h K : 查找给出的按键的在线文档
C-h S : 查找给出的符号的在线文档

C-h l : 显示最后 300 个输入的键
C-h n | C-h C-n : 显示最近版本的变更说明

C-h C-f : FAQ
C-h C-p : 已知的问题（错误）
C-h C-e : 额外的包和关于 Emacs 的信息
C-h C-d : 调试说明
C-h C-o : 发行信息
C-h C-t : TODO 列表

C-M-s | C-u C-s : 正则增量查找

C-u M-; : 删除注释
M-;     : 插入注释
M-\     : 删除光标当前位置到第一个非空白字符的空白
M-r     : 使光标在屏幕中央、顶部和底部循环移动

M-|     : 在选中的区域中运行 shell 命令
M-!     : 运行 shell 命令
